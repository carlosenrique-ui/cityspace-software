# Globe-CitySpace Open MVP

Objetivo: criar uma frente paralela ao IPT-CitySpace para navegação urbana 3D aberta, seleção de área, geração de grid 16x8 e exportação para mesa virtual.

## Fases

G0 - Esqueleto isolado do projeto  
G1 - Tela CesiumJS com OpenStreetMap  
G2 - Busca / fly-to São José dos Campos  
G3 - Seleção de área  
G4 - Grid 16x8  
G5 - Alturas estimadas  
G6 - Exportação JSON/CSV para mesa virtual
