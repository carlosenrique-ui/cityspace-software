## Main responsibilities

- Load scientific grid products generated by the offline pipeline.
- Render the virtual 16 x 8 table.
- Display building/top height values as virtual pin heights.
- Display IPT base/watermark imagery.
- Display grid, contours, and cotas.
- Simulate zigzag traversal over the table cells.
- Provide virtual/real mesa mode switch.
- Provide visual scale mode for 1 x 1 cm and 2 x 2 cm cells.
- Provide cartographic scale indication.
- Preserve the frozen reference for later physical table integration.
